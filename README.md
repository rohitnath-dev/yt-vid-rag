# AI Learning Assistant

An AI assistant trained on YouTube videos and study content using RAG.

## What it does
- Answers questions from video content
- Explains concepts in a simple, student-friendly way
- Uses custom knowledge instead of generic responses

## Tech Stack
- LangGraph
- FAISS
- Sentence Transformers
- OpenRouter API
